# AI News Agent 🤖

An AI-powered news intelligence system that delivers daily insights on OpenAI, Anthropic, and AI trends.

## 🚀 Features
- Real-time AI news (Google News RSS)
- AI-generated summaries
- “Why it matters” insights
- Trend of the Day
- Clean email UI

## 🧠 How it works
1. Fetches latest AI news from RSS
2. Uses OpenAI to summarize + generate insights
3. Formats into a structured email
4. Sends daily digest via Google Apps Script

## 🛠 Tech Stack
- Google Apps Script
- OpenAI API
- RSS feeds

## 📩 Output
Daily AI newsletter with:
- Top story
- Key insights
- Actionable takeaways
